Rating Predictor

Given a prompt this generates generates iamges that the user rates by clicking on a star. The generator can be a Stable Diffusion server or DALL-E (requires an API key from OpenAi.com).

Machine learning

After you have rated several images (100 works well, maybe less) you can train the computer to predict how you would rate a new images you've never seen.

Keyboard shortcuts

  • Learning (when you think you have rated enough and want to start training (or re-start training from scratch))
  • Additional learning (when you have already done 'Learning' but want to do more training)
  • Rating (after training has finished you can rate either the best or worst of a number of generated images)
  • Search (repeats training with different size models and techniques)
  • Inspect (opens interface where you see training progress, do experiments, and save your model)
