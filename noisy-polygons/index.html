Noisy Polygons

A noisy polygon repeatedly goes forward a random amount, turns a random amount, sets the red, green, and blue parts of the pen colour to random values, and sets the pen size to a random value. All random numbers are given by a minimum and maximum value. So there are 12 numbers which control each image: minimum step, maximum step, minimum turn, maximum turn, minimum red, maximum red, minimum green, maximum green, minimum blue, maximum blue, minimum pen size, and maximum pen size. They are called noisy because "noise" (or randomness) is added to the average value for steps, turns, and pen characteristics.

Machine learning

After you have rated several images (100 works well, maybe less) you can train the computer to predict how you would rate a new noisy polygon you've never seen. See these project ideas to explore this further.

Keyboard shortcuts

  • Learning (when are first starting or want to start over again)
  • Additional learning (when you have already done 'Learning' but want to do more)
  • Rating (after learning you rate either the best or worst of a collection of noisy polygons)
  • Search (repeats learning with different size models and techniques)
  • Inspect (opens interface where you see training progress, do experiments, and save your model)
