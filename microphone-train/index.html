  1. Train by clicking a click to train button and then saying the word (or making a sound).
  2. Train each word or phrase multiple times.
  3. Test by pressing the start testing button and say a word already trained.

Don't worry, we aren't sending any audio recordings to a remote server. All the machine learning is being done locally on device, and you can check out our source code on Github.
