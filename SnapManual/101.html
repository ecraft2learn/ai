101 XI. Metaprogramming The scripts and custom blocks that make up a program can be examined or created by the program itself. A. Reading a block The denition of block takes a custom block (in a ring, since it’s the block itself that’s the input, not the result of calling the block) as input and reports the block’s deﬁnition, i.e., its inputs and body, in the form of a ring with named inputs corresponding to the block’s input names, so that those input names are bound in the body. The split by blocks block takes any expression or script as input (ringed) and reports a list representing a syntax tree for the script or expression, in which the ﬁrst item is a block with no inputs and the remaining items are the input values, which may themselves be syntax trees.
