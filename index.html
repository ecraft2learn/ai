Enabling children to build AI programs

The eCraft2Learn project is developing a set of extensions to the Snap! programming language to enable children (and non-expert programmers) to build AI programs.

A guide that describes the extensions, projects, and the larger context currently consists of four chapters:

  1. Speech Output        Teacher guide Student guide
  2. Speech Input          Teacher guide Student guide
  3. Image Recognition Teacher guide Student guide
  4. Machine Learning   Teacher guide Student guide
An early version of the extension is described in the paper Child-friendly Programming Interfaces to AI Cloud Services by Ken Kahn and Niall Winters.

The following demonstration programs use the AI Snap! blocks. They run best in the Chrome browser.

  1. Speak with random pitch, rate, voice, and language  (requires speaker/headphones)
  2. Speak commands to a sprite  (requires microphone and speaker/headphones)
  3. Funny templated sentences  (requires microphone and speaker/headphones)
  4. Story generator  (requires microphone and speaker/headphones)
  5. Talk to Wikipedia  (requires microphone and speaker/headphones)
  6. Says what it sees  (requires webcam and API key(s))
  7. Says what it sees when asked to  (requires microphone, speaker/headphones, webcam and API key(s))
  8. After training the turtle moves depending on which way your finger is pointed  (requires webcam)
  9. After training the turtle moves left or right depending on which way you lean  (requires webcam)
