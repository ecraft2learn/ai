An explanation of the how the AI Question and Answer Snap! project works

The project loads two similar models that were trained to produce one of six labels given a sentence. The models were generated with the same program but with a different split of the data between training and validation. The average of their confidences that each label is correct is used to answer questions.

The models were the result of five hyperparameter searches. Each search explored twenty variants and then replaced the current model with the best one found if it is better. See the search and replace block to see how this was implemented.

The project splits the data into training, validation, and test datasets in such a way that they each contain the same number of each category of question. See the create training and validation data block.

The project turns sentences into lists of 512 numbers using the features of sentence block. This relies upon the universal sentence encoder.
