Instead of "manually" searching for optimal values for hyperparameters such the learning rate, optimization method, number of layers, etc. the search can be automated. The idea is to create, train, and evaluate the model with different hyperparameter values to find the one with the lowest error in its predictions. For optimization methods and loss functions it chooses randomly among the options. For the number of training iterations (sometimes called "epochs") it picks random numbers between one half and double the current setting. For number of layers it generates six variants of the current setting:

  1. increase the size of each layer by at most a factor of 2 (except last one which is the output)
  2. decrease the size of each layer by at most a factor of 1/2
  3. insert a new layer of the same size as the first layer
  4. insert a new layer up to double the size of the first layer
  5. remove the first layer (unless it is the only remaining hidden layer)
  6. keep the current list of layers

You can use the settings to control which hyperparameters are searched and how many searches to perform.
